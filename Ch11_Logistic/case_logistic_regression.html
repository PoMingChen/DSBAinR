資料建模與分析

1. 考量「行銷因素」的邏輯迴規模型

首先，我們先建立只考量「行銷因素」的邏輯迴歸模型。在 R 語言中，我習慣使用內建的 glm 函數 (glm = generalized linear model)，主要要填入的參數有：

  • 模型公式：假設你的想建立的邏輯迴歸模型反應變數為 \(Y\)，解釋變數為 \(X_1\) 與 \(X_2\)，可以輸入 Y ~ X1 + X2。
  • 資料集合：輸入 data = dataset.name 即可，記得變數的名稱要與模型公式中的名稱完全符合。
  • 模型種類：由於 glm 不只適用於邏輯回歸，如果你希望建立邏輯迴歸模型，請使用 family = binomial(link="logit")。
# Build logistic regression model
marketing.model <- glm(is_loyal ~ dm_message + dm_post + dm_email +              
                         credit_card_vendor + credit_card_bonus + 
                         tv_ad + youtube_ad_1 + youtube_ad_2 + youtube_ad_3, 
                       data = complete.data, family = binomial(link="logit"))

我們可以使用 summary 函數輸出各個變數的係數估計與 p-值：

  • 對客戶忠誠度有顯著正面影響 (\(\alpha = 0.05\)) 的變數：
    dm_post, dm_email, credit_card_bonus3, credit_card_bonus4, tv_ad, youtube_ad_2
  • 對客戶忠誠度有顯著負面影響 (\(\alpha = 0.05\)) 的變數：
    dm_message, youtube_ad_1

問題：這個模型可以完整解釋客戶的忠誠與否嗎？（答：顯然不行，缺少重要面向。）

接下來，我們利用 predict(model, data, type) 函數得到模型估計的\(\mathbb{P}(Y=1 | X_1,\cdots,X_p)\)：

  • type = "response"：得到每一個個體的機率。
  • type = "term"：得到每一個個體 \(\beta_i\cdot X_i\) 的數值。
predict.prob <- predict(marketing.model, complete.data, type = "response")

接下來，利用 InformationValue 套件中的函式，可已得到使分類誤差最小的 cutoff probability，以及計算常用的分類好壞指標：

  • 模糊矩陣 (confusion matrix)

  • \(\mathrm{precision}=\frac{TP}{TP+FN}\) (真正的 Positive 有多少被分類正確)

  • \(\mathrm{sensitivity~(recall)}=\frac{TP}{TP+FN}\) (被分類成 Positive 有多少是真正的 Positive)

  • \(\mathrm{specificity}=\frac{TN}{TN+FP}\) (被分類成 Negative 有多少是真正的 Negative)
# install.packages("InformationValue")
library("InformationValue")

# Calculate cut-off probability minimizing the miscalssification error
opt.cutoff <- optimalCutoff(complete.data$is_loyal, predict.prob)[1] 

# Calculate Important Assessment Measure
confusionMatrix(complete.data$is_loyal, predict.prob, threshold = opt.cutoff)

最後，我喜歡利用 plotROC 套件繪製分類模型的 ROC 曲線，並計算 AUC (Area Under Curve)。模型的 AUC 為 0.837，分類的準確度還算不錯。

# devtools::install_github("sachsmc/plotROC")
library("plotROC")

predict.table <- data.frame(true_label = complete.data$is_loyal,
                            predict_prob = predict.prob)

# Plot ROC curve and calculate AUC
basic.plot <- ggplot(predict.table, aes(d = true_label, m = predict.prob)) +
  geom_roc(n.cuts = 3, labelsize = 3, labelround = 2)
basic.plot + style_roc() +
  annotate("text", x = .75, y = .25, size = 5,
           label = paste("AUC =", round(calc_auc(basic.plot)$AUC, 3)))

由於行銷變數都是 0 / 1 的虛擬變數 (dummy variable)，scale 是一致的，所以我們可以直接比較邏輯迴歸中細述的大小，而係數大小代表的是「每個變數對於客戶忠誠度 odds 的影響程度」。

我喜歡用橫向的 Bar Chart 去視覺化迴歸係數，大家可以參考以下程式碼。值得注意的是，DM 相關的行銷策略表現在「同時考量服務品質與行銷策略」的模型 3 中時在表現得太差，所以我並不一定會真得呈現這個結果，或是會用稍微委婉的方式去說明這個結果。
